Pharmacy Management — Windows Desktop App

This repository contains a Windows Forms (.NET Framework 4.7.2) app. Vercel is a web hosting platform and cannot run desktop applications. Instead, this page provides a safe download and setup instructions.

Download

⬇️ Download for Windows (.zip) Direct .exe (advanced)

If the direct .exe fails to launch, use the .zip which includes dependencies.

How to run

  • Requires Windows with .NET Framework 4.7.2 Developer Pack.
  • Extract the pharmacy-winforms.zip anywhere.
  • Open the folder and run PHARMACY 1.exe.
  • Configure SQL connection in PHARMACY 1.exe.config if needed.
Database schema (SQL) Repo README

Why doesn’t it “run” on Vercel?

Vercel hosts static sites and serverless web apps. Windows desktop apps need a Windows OS and cannot run in Vercel’s Linux web runtime. This repo is correctly deployable as a static site that offers downloads and documentation.

If you need a web version, consider porting to ASP.NET (Web) or Blazor, or building a separate web frontend.
